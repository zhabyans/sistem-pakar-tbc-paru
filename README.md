# Sistem Pakar TBC Paru

Aplikasi untuk mendeteksi penyakit paru sejak dini menggunakan Metode Certainty Factor dan Logika Fuzzy

## Installation

Use the package manager [npm](https://nodejs.org/en/) to install npm.

```bash
npm install
ionic serve
```
